Robo Control

Robo Control is an App used to wirelessly control a robot. You control it either through a virtual joystick or by use of the Gesture Control bracelet MYO made by Thalmic Labs.

To use Robo Control to handle a robot, the smartphone has to be connected to the WiFi-Network generated by, for example, an XBee® WiFi-Module made by Digi International.
If a valid connection is established, Robo Control sends relevant data, which will be analysed and processed by a microprocessor, to the WiFi-Module up to ten times each second.

To use MYO for controlling the robot, Bluetooth Low Energy (BLE) is required.
Find further details under Control.

First Steps

  • Connect WiFi

    Connect your smartphone to the WiFi-Module.

    Robo Control -> WiFi Settings
    or
    Settings (Smartphone) -> pair WLAN and MYO

  • Pair MYO

    Pair MYO with your smartphone via Bluetooth LE.

    Robo Control -> BT Settings
    or
    Settings (Smartphone) -> Bluetooth

  • Control

    Start Robo Control and adjust the settings, if necessary. Choose your desired control-mode.

    If you are using the Gesture Control-Mode, select the bracelet in the next dialogue.
